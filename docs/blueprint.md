# **App Name**: Arkitekturopprøret Bergen

## Core Features:

- Public Website: Public-facing website showcasing Arkitekturopprøret's vision, news, examples of architecture, and a contact form.
- Admin Dashboard: Admin dashboard for managing website content, including news, image galleries, member profiles, and social media planning.
- Board Member Profiles: Individual profile pages for each board member with their bio, contact info, and contributions to the website.
- Architectural Project Database: Research database where board members can log, comment on, and categorize architectural projects in Bergen as "beautiful" or "ugly."
- Social Media Planner: Social media planning tool for scheduling posts, assigning responsibilities, and tracking post performance.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F), reminiscent of classic Bergen architecture, to convey stability and tradition.
- Background color: Light beige (#F5F5DC) to ensure legibility of content.
- Accent color: Muted gold (#BDB76B) to highlight calls to action.
- Headline font: 'Playfair', a serif font lending a classic and elegant appeal to headers and titles; body font: 'PT Sans' (sans-serif), ensuring the main text content is easy to read.
- Use simple, classic icons related to architecture, community, and Bergen.
- Clean, structured layout with clear navigation to provide easy access to all sections.
- Subtle transitions to load new content and reveal details on architectural projects.
